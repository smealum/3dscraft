3DSCraft

3DSCraft is a voxel game in style of the popular game "Minecraft" created for the Nintendo 3DS. It is currently in a very early stage and will become more complex in the future.

Features

  • Infinity real time generating worlds
  • Full creative mode with a growing amount of Blocks to choose from
  • A touch screen UI
  • With even more
